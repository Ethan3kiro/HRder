# 大屏模式布局修复说明

修复时间: 2026年7月15日

## 🎯 修复目标

解决大屏模式（全屏数据录入窗口）的滑动窗口兼容性问题：
- ❌ 滑动窗口在某些设备上失效
- ❌ 有时会闪退
- ❌ 底部按钮需要滑动才能看到
- ❌ 右侧内容被遮挡

## 🔧 解决方案

### 核心改动

**从上下滚动布局改为左右固定布局**

#### 修复前
```
┌─────────────────────────────────────┐
│ 顶部工具栏（很长，包含所有控件）      │
├─────────────────────────────────────┤
│ 左侧图片 (40%)  │  右侧表格 (60%)   │
│                │                   │
│ 需要滚动 ↕     │  需要滚动 ↕       │
│                │                   │
│ [按钮被遮挡]    │  [内容被遮挡]      │
└─────────────────────────────────────┘
```

#### 修复后
```
┌─────────────────────────────────────┐
│ 简化标题栏 + 返回按钮                │
├─────────────────────────────────────┤
│ 左侧面板 (40%)  │  右侧面板 (60%)   │
│ ┌────────────┐  │  ┌─────────────┐ │
│ │1. 选择文件  │  │  │4. 数据表格  │ │
│ │2. 图片预览  │  │  │   (内置滚动)│ │
│ │3. 识别按钮  │  │  │5. 保存区域  │ │
│ │   无滚动    │  │  │   月份      │ │
│ └────────────┘  │  │   设备      │ │
│                │  │   [保存按钮] │ │
│                │  └─────────────┘ │
└─────────────────────────────────────┘
```

## 📝 详细修改

### 1. 布局结构调整

**文件**: `src/gui/widgets/fullscreen_data_entry.py`

#### 主窗口布局
```python
# 修复前：使用滚动区域
content_layout = QHBoxLayout()
left_panel = self.create_image_panel()  # 包含滚动区域
right_panel = self.create_data_panel()  # 包含滚动区域

# 修复后：左右固定面板，无滚动区域
content_layout = QHBoxLayout()
left_panel = self.create_left_panel()   # 固定布局
right_panel = self.create_right_panel() # 固定布局
```

#### 宽度比例调整
- 左侧面板: 45% → **40%** (减少5%)
- 右侧面板: 55% → **60%** (增加5%)
- 理由：数据表格需要更多空间

### 2. 工具栏简化

#### 修复前（工具栏过长）
- 返回按钮
- 文件选择 + 浏览按钮
- 模板识别按钮
- API识别按钮
- 手动录入按钮
- 月份输入
- 设备选择
- 保存按钮
- 进度条

#### 修复后（精简）
- 标题文字
- 进度条
- 返回按钮

**移动位置**:
- 文件选择 → 左侧面板
- 识别按钮 → 左侧面板
- 月份/设备/保存 → 右侧面板

### 3. 左侧面板设计

```
┌─────────────────┐
│ 1. 选择图片文件  │
│   [文件路径]     │
│   [📁 浏览]     │

│ 2. 图片预览     │
│   ┌───────┐    │
│   │ 图片  │    │
│   │ 显示  │    │
│   └───────┘    │

│ [识别结果提示]  │
├─────────────────┤
```

**特点**:
- ✅ 分步骤布局，清晰直观
- ✅ 按钮大而明显
- ✅ 无滚动区域，避免兼容性问题
- ✅ 图片自适应显示

### 4. 右侧面板设计

```
┌─────────────────┐
│ 4. 填写/核对数据│
│ ┌─────────────┐ │
│ │ 数据表格    │ │
│ │ (内置滚动条)│ │
│ │ 71行数据    │ │
│ └─────────────┘ │
├─────────────────┤
│ 5. 保存数据     │

```

**特点**:
- ✅ 表格使用内置滚动条（QTableWidget自带）
- ✅ 列宽优化：数值100px，单位50px
- ✅ 月份和设备分两行显示
- ✅ 保存按钮固定在底部可见
- ✅ 增加右边距避免遮挡

### 5. 边距和间距优化

```python
# 主布局边距
main_layout.setContentsMargins(10, 10, 20, 10)  # 右边距增加到20

# 内容布局间距
content_layout.setSpacing(15)  # 左右面板间距增加

# 右侧面板边距
panel_layout.setContentsMargins(10, 15, 15, 10)  # 右边距15

# 表格列宽
self.data_table.setColumnWidth(1, 100)  # 数值列: 120→100
self.data_table.setColumnWidth(2, 50)   # 单位列: 60→50
```

## 🔧 技术细节

### 关键代码改动

#### 1. 窗口初始化
```python
# 启动时自动最大化
self.showMaximized()

# 设置窗口标志（Windows兼容）
self.setWindowFlags(
    Qt.WindowType.Window |
    Qt.WindowType.WindowMaximizeButtonHint |
    Qt.WindowType.WindowCloseButtonHint
)
```

#### 2. 布局比例
```python
# 左右面板宽度比例
left_panel = self.create_left_panel()
content_layout.addWidget(left_panel, 40)  # 40%

right_panel = self.create_right_panel()
content_layout.addWidget(right_panel, 60)  # 60%
```

#### 3. 表格设置
```python
# 使用表格内置滚动条
self.data_table.setVerticalScrollBarPolicy(
    Qt.ScrollBarPolicy.ScrollBarAsNeeded
)
self.data_table.setHorizontalScrollBarPolicy(
    Qt.ScrollBarPolicy.ScrollBarAsNeeded
)
```
