# 模板识别对话框布局修复说明

修复时间: 2026年7月15日

## 🎯 修复目标

解决点击"模板识别"按钮后进入的图片对齐对话框的滑动窗口问题：
- ❌ 滑动窗口在某些设备上失效
- ❌ 有时会闪退
- ❌ 底部按钮需要滑动才能看到
- ❌ 右侧内容可能被遮挡

## 🔧 解决方案

### 核心改动

**从上下滚动布局改为左右固定布局**

## 📝 详细修改

### 1. 布局结构调整

**文件**: `src/gui/widgets/template_ocr_dialog.py`

#### 主要改动

```python
# 修复前：使用滚动区域
scroll = QScrollArea()
scroll_content = QWidget()
scroll_layout = QVBoxLayout(scroll_content)
scroll_layout.addWidget(self.alignment_widget)
scroll_layout.addWidget(controls)
scroll.setWidget(scroll_content)

# 修复后：左右固定布局
content_layout = QHBoxLayout()
content_layout.addWidget(self.alignment_widget, 70)  # 左侧70%
content_layout.addWidget(right_panel, 30)  # 右侧30%
```

## 🔧 技术细节

### 关键改动点

#### 1. 移除滚动区域
```python
# 删除
scroll = QScrollArea()
scroll.setWidgetResizable(True)
scroll.setHorizontalScrollBarPolicy(...)

# 改为
content_layout = QHBoxLayout()  # 直接使用水平布局
```

#### 2. 创建右侧面板
```python
def _create_right_panel(self) -> QWidget:
    # 垂直布局包含多个分组
    panel_layout = QVBoxLayout()
    
    # 添加各个分组
    panel_layout.addWidget(offset_group)
    panel_layout.addWidget(scale_group)
    panel_layout.addWidget(actions_group)
    panel_layout.addWidget(tip_label)
    panel_layout.addStretch()  # 底部留白
```

#### 3. 偏移标签格式
```python
# 修复前：单行显示
"X: +0.5  Y: -1.0"

# 修复后：两行显示
"X: +0.5\nY: -1.0"
```

## 🚀 使用方法

### 启动测试
```bash
# 测试新布局
python3 test_template_dialog.py

# 或通过主程序
python3 main_gui.py
# 选择图片 → 点击"🖼️ 模板识别"
```

### 操作流程
1. 对话框打开，图片自动加载
2. 使用方向键调整图片位置
   - 方向键：移动1像素
   - Shift+方向键：移动0.5像素
3. 使用滑块调整缩放（98%-102%）
4. 点击"🧪 测试识别"查看效果
5. 调整满意后，点击"✅ 识别并填入"
