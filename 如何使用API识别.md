# 如何使用 API 识别功能

## 🎉 好消息

API 识别功能已经集成到程序中！使用云端 AI 模型，无需训练本地模型。

## ⚙️ 首次配置（重要）

### 第一步：配置 API 密钥

**📍 配置文件位置**: `config/api_config.json`

**🔧 如何配置**:

1. 打开 `config/api_config.json` 文件
2. 找到 `api_key` 字段
3. 将值替换为您的阿里百炼 API 密钥
4. 保存文件并重启程序

**示例**:
```json
{
  "api_key": "sk-your-actual-api-key-here"
}
```

**📖 详细配置说明**: 请查看 [API配置说明.md](API配置说明.md)

### 第二步：获取 API 密钥

如果您还没有 API 密钥：

1. 访问 [阿里云百炼平台](https://bailian.console.aliyun.com/)
2. 注册/登录账号
3. 创建 API 密钥
4. 复制密钥到配置文件

**详细步骤**: 参考 [API配置说明.md](API配置说明.md) 中的"如何获取阿里百炼 API 密钥"章节

## 📋 快速开始

### 1. 启动程序

```bash
python3 main_gui.py
```

或者在 Windows 上双击：
```bash
start_gui.bat
```

### 2. 使用 API 识别

#### 方法 A：普通模式

1. 点击左侧菜单的"📥 数据录入"
2. 点击"📁 浏览"选择截图文件
3. 点击"🌐 API 识别"按钮（新增的按钮）
4. 等待 15-30 秒，识别完成
5. 查看识别结果，核对并修正错误
6. 填写月份和设备
7. 点击"💾 保存到数据库"

#### 方法 B：全屏模式

1. 点击左侧菜单的"📥 数据录入"
2. 选择截图文件
3. 点击"🖥️ 全屏模式"
4. 在全屏界面点击"🌐 API"按钮
5. 等待识别完成
6. 核对并保存

## 💡 使用建议

### 推荐场景

1. **新手用户**: 使用 API 识别，无需训练模型
2. **偶尔使用**: API 识别最方便
3. **数据敏感**: 使用手动录入或训练本地模型
4. **大量使用**: 训练本地模型更经济

### 最佳实践

1. **首次使用**: 先用 API 识别体验功能
2. **核对数据**: 识别后务必核对原始截图
3. **成本控制**: 记录使用次数，避免超支
4. **备用方案**: 如果 API 不可用，使用手动录入

## 🐛 常见问题

### Q1: API 识别按钮不可用？
**A**: 检查 `config/api_config.json` 文件是否存在，重启程序。

### Q2: 识别超时？
**A**: 检查网络连接，或稍后重试。

### Q3: 识别结果不准确？
**A**: 
- 确保截图清晰
- 手动修正错误的数据
- 反馈问题以改进提示词

### Q4: 想要更快的识别速度？
**A**: 考虑训练本地深度学习模型，参考 `QUICK_START_DL_TRAINING.md`
