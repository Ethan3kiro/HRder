# 阿里云百炼（通义千问）配置指南

## 🎯 为什么选择阿里云百炼？

- **强大的 OCR 能力** - qwen-vl-plus 对文字识别准确率高
- **支持高分辨率** - 支持超高像素和任意宽高比
- **国内服务** - 访问速度快，稳定性好
- **价格合理** - 按量计费，性价比高

## 📝 获取 API Key

### 1. 注册阿里云账号

访问：https://www.aliyun.com/

使用手机号或支付宝注册。

### 2. 开通百炼服务

1. 访问：https://bailian.console.aliyun.com/
2. 点击"立即开通"
3. 同意服务协议

### 3. 创建 API Key

1. 进入百炼控制台
2. 点击右上角头像 → API-KEY 管理
3. 点击"创建新的 API-KEY"
4. 复制生成的 API Key（格式类似：`sk-xxxxxxxxxxxxxxxxxxxxxxxx`）

⚠️ **重要**：API Key 只显示一次，请妥善保存！

## 🔍 查找可用模型

运行测试脚本查找当前可用的视觉模型：

python3 test_aliyun_models.py

## ⚙️ 配置方法

### 方法 1：通过设置页面配置（推荐）

1. 启动 HarrisReader
2. 进入"系统设置"页面
3. 在"API 识别设置"部分：
   - **API Key**：粘贴你的阿里云百炼 API Key
   - **API URL**：`https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions`
   - **模型名称**：`qwen-vl-plus`（或测试脚本找到的其他模型）
   - **Max Tokens**：留空（不填写）
4. 点击"保存 API 配置"
5. 点击"测试 API 连接"验证配置
6. 重启程序使配置生效

### 方法 2：手动编辑配置文件

编辑 `config/api_config.json`：

```json
{
  "provider": "custom",
  "api_key": "你的阿里云百炼API密钥",
  "api_url": "https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions",
  "extra_params": {
    "headers": {
      "Content-Type": "application/json"
    },
    "auth_header": "Authorization",
    "auth_prefix": "Bearer",
    "model": "qwen-vl-plus",
    "payload_template": {
      "model": "qwen-vl-plus",
      "messages": [
        {
          "role": "user",
          "content": [
            {
              "type": "text",
              "text": "识别图中所有数据:\n1.COMBINER:AZ,BZ,CZ,DZ,AB,CD,ABCD(单位°C)\n2.Z-Plane A/B/C/D:每个8行,每行Current(A)和ISO Temp(°C)\n格式:Z-Plane A-Current-1\n\n返回JSON:\n{\"data\":[{\"item_name\":\"AZ\",\"value\":30.0,\"unit\":\"°C\"},{\"item_name\":\"Z-Plane A-Current-1\",\"value\":7.2,\"unit\":\"A\"}]}\n\n要求:纯JSON,不用markdown,value为数字,全部71项"
            },
            {
              "type": "image_url",
              "image_url": {
                "url": "data:image/jpeg;base64,{{image}}"
              }
            }
          ]
        }
      ]
    },
    "data_path": "choices.0.message.content",
    "timeout": 120
  }
}
```

## ❓ 常见问题

### Q1: API Key 在哪里找？

A: 登录 https://bailian.console.aliyun.com/ → 点击头像 → API-KEY 管理 → 创建新的 API-KEY

### Q2: 提示"API 连接失败"怎么办？

A: 检查以下几点：
1. API Key 是否正确（格式：`sk-...`）
2. 网络连接是否正常
3. API URL 是否正确：`https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions`
4. 是否已在百炼控制台开通服务

### Q3: 哪个模型最适合这个项目？

A: **推荐 qwen-vl-plus**：
- OCR 能力强，适合识别发射机截图
- 支持高分辨率和任意宽高比
- 价格合理，性价比高

### Q4: 识别速度慢怎么办？

A: qwen-vl-plus 通常在 10-30 秒内完成识别。如果超过 1 分钟：
- 检查网络连接
- 尝试更换网络
- 检查图像大小（程序会自动压缩）

### Q5: 识别准确率如何？

A: 根据测试，qwen-vl-plus 对清晰的截图识别准确率可达 95%+。建议：
- 使用高清截图
- 确保文字清晰可见
- 避免截图过暗或过亮

### Q6: 有使用限制吗？

A: 阿里云百炼按量计费，可能有以下限制：
- 每分钟请求次数限制（通常足够个人使用）
- 单次图片大小限制（程序会自动压缩）
- 账户余额不足时无法调用

### Q7: Max Tokens 应该设置多少？

A: **建议留空（不设置）**。阿里云的模型会自动处理输出长度。如果设置了 `max_tokens`，可能会导致：
- 输出被截断
- 或者 API 返回 400 错误

### Q8: 和智谱AI相比有什么优势？

A: 
- ✅ **支持更长的输出** - 不受 1024 tokens 限制
- ✅ **OCR 能力更强** - 专门优化了文字识别
- ✅ **支持高分辨率** - 可以处理更大的图像
- ✅ **更稳定** - 企业级服务，稳定性好
