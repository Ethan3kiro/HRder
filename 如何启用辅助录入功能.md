# 如何启用辅助录入功能

## 问题

在数据录入页面，"🤖 辅助录入"按钮是灰色的，无法点击。

## 原因

辅助录入功能需要深度学习模型支持，需要：
1. 安装 PyTorch
2. 训练模型

## 解决方案

### 第一步：安装 PyTorch

#### 方法 1：使用自动安装脚本（最简单）

1. 双击运行项目根目录下的 `install_pytorch_windows.bat`
2. 等待安装完成
3. 看到"PyTorch 安装成功！"提示

#### 方法 2：手动安装

打开命令提示符，运行：

```bash
pip install torch torchvision --index-url https://download.pytorch.org/whl/cpu
```

### 第二步：训练模型

安装 PyTorch 后，需要训练模型：

```bash
# 查看详细训练指南
type QUICK_START_DL_TRAINING.md

# 快速开始训练
python training/tools/train_dl_model.py
```

### 第三步：验证

1. 重启程序：`python main_gui.py`
2. 进入"数据录入"页面
3. 将鼠标悬停在"🤖 辅助录入"按钮上
4. 如果仍然无法点击，查看提示信息

## 常见问题

### Q1: 安装 PyTorch 失败

**错误信息：** "No matching distribution found for torchvision"

**解决方案：**
```bash
# 使用官方 CPU 版本源
pip install torch torchvision --index-url https://download.pytorch.org/whl/cpu
```

### Q2: 没有训练数据

**提示：** "模型文件不存在"

**解决方案：**
1. 准备训练数据（发射机截图）
2. 使用标注工具标注数据
3. 运行训练脚本

详细步骤请参考：`QUICK_START_DL_TRAINING.md`

### Q3: 按钮仍然无法点击

**检查步骤：**
1. 将鼠标悬停在按钮上，查看具体原因
2. 确认 PyTorch 已安装：
   ```bash
   python -c "import torch; print(torch.__version__)"
   ```
3. 确认模型文件存在：
   - `models/digit_ocr_model.pth`
   - `models/coordinates.json`

## 不想使用辅助录入？

如果不需要深度学习辅助识别，可以直接使用：
- **✍️ 手动录入**：完全手动填写所有数据
- **🖥️ 全屏模式**：在更大的窗口中查看截图并手动录入

这两个功能不需要安装 PyTorch。
